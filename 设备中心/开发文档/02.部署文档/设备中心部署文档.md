## 一、环境准备

### 1.1 服务器配置要求
- 操作系统：Debian GNU/Linux 12 (bookworm)
- Node.js：v20.18.0
- PM2：5.4.3

## 二、PM2 配置与启动

### 2.1 启动服务
```bash
# 在项目目录执行
pm2 start ecosystem.config.js

# 查看进程状态
pm2 status
```

### 2.2 设置开机自启
```bash
# 生成开机启动命令
pm2 startup

# 保存当前进程列表
pm2 save
```
