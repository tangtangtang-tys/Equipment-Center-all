# 查询设备信息（全部）

> 钉钉节点：Y1OQX0akWm3Z4l3xsvwkaoPKJGlDd3mE

### **接口名称：查询设备信息**

#### **接口地址**
```
POST /deviceCenter/find/info
```

测试环境地址：http://192.168.252.124:28627[/iot\_device\_center/deviceCenter/find/info](https://api-asset03.eye4cloud.com/iot_device_center/deviceCenter/find/info)

测试环境走运维网关：http://192.168.252.115:2123[/iot\_device\_center/deviceCenter/find/info](https://api-asset03.eye4cloud.com/iot_device_center/deviceCenter/find/info)

线上地址走运维网关：[https://api-asset03.eye4cloud.com/iot\_device\_center/deviceCenter/find/info](https://api-asset03.eye4cloud.com/iot_device_center/deviceCenter/find/info)

线上地址：[http://192.168.1.189:28627/deviceCenter/find/info](https://api-asset03.eye4cloud.com/iot_device_center/deviceCenter/find/info)

#### **请求头（Headers）**

| 参数名 | 值示例 | 是否必填 | 说明 |
|---------|---------|------------|------|
| Content-Type | application/json | 是 | 固定值 |

#### **请求体（Body）**
```json
{
  "did": "VQHRGX0260442GSOH"
}
```

**参数说明**

| 字段名 | 类型 | 是否必填 | 说明 | 示例值 |
|---------|------|------------|------|---------|
| did | string | 是 | 设备id | VQHRGX0260442GSOH |

#### **响应字段说明**

| 字段名 | 类型 | 说明 |
|---------|------|------|
| id | string | 用户唯一标识 |
| username | string | 用户名 |
| email | string | 用户邮箱 |
| created\_at | timestamp | 用户创建时间（ISO 8601） |
| error | string | 错误类型（如适用） |
| message | string | 错误描述（如适用） |
