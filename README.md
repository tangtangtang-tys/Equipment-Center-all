# Equipment Center

设备中心平台原型，用于复刻并完善第一阶段设备查询、设备详情、日志、设备状态、物联网卡和设备地图等交互流程。

## 本地运行

```bash
npm install
npm run dev
```

默认访问地址：

```text
http://localhost:5173/
```

## 构建

```bash
npm run build
```

## 目录

- `src/`：前端页面、交互和模拟数据
- `设备中心/`：从钉钉知识库导出的项目文档
- `download_dingtalk_device_center.sh`：知识库导出脚本
